# Linksly Social

## linksly.social

Saas product similar to _linktr.ee_ to post links with aditional feature of adding blogs, sharing resources and photos with stripe auth

### FEATURES

- Stripe
- Auth
- Security
- Analitics
- DragNDrop
- more added in future

### Instalation

```.ps
git clone
```

```.ps
pnpm dev
```

```.node
host on vercel
```
